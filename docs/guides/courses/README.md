# Courses

Courses are project-based guides that teach CRK by building a real local case
workspace. They are documentation only: raw source downloads and generated case
records stay under ignored `data/cases/` paths.

| Course | Status | Focus |
| --- | --- | --- |
| [MKUltra source-traceable course](mkultra/) | Draft, source pack captured | Greenfield install, source preservation, skill/MCP/CLI workflows, controversy boundaries. |

## Course Rules

- Treat every public-facing statement as a claim that needs a source ID and a
  locator.
- Keep raw source files out of git. Store them under `data/cases/<case>/raw/`.
- Teach contested material as contested. Do not promote testimony, rumors,
  forum posts, or metadata-only captures into established facts.
- Add screenshot files only after the relevant command and UI state can be
  reproduced.
