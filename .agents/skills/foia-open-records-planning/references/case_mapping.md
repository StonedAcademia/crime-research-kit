# TRCR Case Mapping For FOIA Open Records Planning

## Planning Artifacts

`plan-open-records` writes a JSON planning artifact under `staging/candidates/`. It is not evidence.

## Sources

Register these as sources when received:

- Request letter.
- Agency acknowledgment.
- Response letter.
- Released records.
- Withholding/exemption log.
- Appeal letter and appeal response.

## Claims

Do not create factual case claims from the request plan. Create claims only from source-backed responsive records or official response letters.

## Events

Use events for request submitted, acknowledgment received, response received, appeal filed, release received, or denial upheld/overturned.
