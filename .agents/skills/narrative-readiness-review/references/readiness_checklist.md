# Narrative Readiness Checklist

- Source IDs present and valid.
- Source grades suitable for public use.
- Source spans present where precise citation is needed.
- Assertion type matches public wording.
- Status/confidence are not overstated.
- Contradictions and denials are represented.
- Source independence is reviewed before calling a claim corroborated.
- Privacy review is clear.
- Public export is safe.
- Quotes are short and source-bound.
