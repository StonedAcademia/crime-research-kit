# CRK Case Mapping For Source Independence Audit

## Reports

`source-independence` writes `exports/source_independence_report.json`.

## Sources

Set `independence_group` when provenance is known. Examples: publisher group, wire service, court docket, press release, archive packet, author, or agency packet.

## Claims

Do not mark a claim `corroborated` solely because dependent sources repeat it. Use notes or follow-up edits to caveat same-source chains.
