# Source Independence Rules

Dependent source patterns:

- Same wire story or syndication chain.
- Same press release or organization statement reposted by multiple outlets.
- Same author, publisher, parent outlet, or editorial package.
- Same court docket, filing packet, exhibit packet, or archive packet.
- Same interview excerpt, documentary clip, transcript, or social post repeated elsewhere.

Use `sources[].independence_group` to record known provenance groups.
