# Geographical Location Source Lanes

Use public-interest sources only. Register each source before extraction.

## Primary/Source-Direct

- Court exhibits, public filings, police/public reports, official maps, agency GIS pages, hearing transcripts, and official incident summaries.
- Original media, captions, photos, videos, and transcripts where location is visible or source-stated.
- Public archives, historical maps, institutional maps, and public facility pages.

## Strong Secondary

- Local reporting that names event scenes, routes, facilities, or recovery/find locations.
- Investigative reporting with maps, exhibits, named sourcing, or official-record references.
- Documentaries or podcasts only when the original transcript/media and timestamps are available.

## Supporting Context

- Gazetteers, historical map collections, public transit/road maps, weather/geography context, and public land-use references.
- Property, parcel, deed, permit, zoning, or ownership records only after routing that portion to `property-location-records`.

## Lead-Only

- Unverified map pins, forum maps, social media, AI summaries, unsourced coordinate lists, and copied timelines.

Lead-only sources can guide searches but cannot establish event location, evidence location, route, or presence.

## Locator Types To Preserve

- Page, paragraph, exhibit number, docket item, timestamp, caption, image frame, map sheet, grid, coordinate, address as printed, route segment, URL fragment, and archive capture.
